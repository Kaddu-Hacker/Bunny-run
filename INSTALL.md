# Installation Guide

## Prerequisites

- Android 6.0+ (API 24+)
- Accessibility Service enabled

## Steps

1. Download the APK from releases
2. Enable "Unknown Sources" in Settings > Security
3. Install the APK
4. Open BunnyBot
5. Enable Accessibility Service when prompted
6. Grant necessary permissions
7. Start the bot

## Troubleshooting

- If bot doesn't work, ensure Accessibility Service is enabled
- Calibrate the path color before running
- Check that the game is in focus
